В данном разрешении используется browserify, так как используется npm модуль docx для создания .docx файлов.
Все изменения проводить в script.js, после чего в консоли надо прописать команду:

browserify script.js -o index.js

Для установки распакованного расширения необходимо зайти в "Расширения" google chrome, включить режим
разработчика и нажать "Загрузить распакованное расширение"
